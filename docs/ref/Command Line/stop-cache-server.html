The Gemfire plugin for Grails provides integration with the [GemFire|http://www.vmware.com/products/application-platform/vfabric-gemfire/overview.html] in-memory distributed data management platform.

(Quick Reference)

stop-cache-server

Purpose

Stops a single cache server instance.

Requires that Gemfire is installed and the GEMFIRE environment variable is set.

Examples

Stops a cache server by name:

grails stop-cache-server server1

Description

Usage:

$ grails stop-cache-server [name]
