cq

Purpose

Enable the use of continuous queries in Gemfire in GORM for Gemfire.

Examples

Person.cq.findAllByLastName("Flintstone") { event ->
			println "GOT EVENT $event"
}

Description

The syntax for executing a continuous query is as follows:

CLASS_NAME.cq.DYNAMIC_FINDER(ARGUMENTS, EVENT_HANDLER)

The last argument is a closure which acts as an event handler whenever a query result is found. The event object is an instance of the CqEvent class.

There are some limitations over normal queries. Refer to the Gemfire documentation on the subject for more information.
